## Scope Notes
- The public paper and repo describe a cybersecurity framework, not a trainable neural model.
- The implementation therefore centers on orchestration, safe tool execution, evaluation, and robotics hooks.
- Private competition assets are out of scope for exact reproduction; PRDs use public/local repo assets where available.
